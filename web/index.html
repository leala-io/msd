MSD Validator

Validates schema conformance against the MSD JSON Schema — not business/semantic validity. (No Güteklassen or assessment logic.)

Runs entirely in your browser; your file is never uploaded, sent, or stored.

A client-side schema-conformance check, in the spirit of the openly hosted validators common across the mobility-data standards landscape (e.g. GBFS). Built on the same JSON Schema (Draft 2020-12) validation core as the reference CLI.
